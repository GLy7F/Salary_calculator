# Architecture - حاسبة الراتب

## نظرة عامة على البنية المعمارية

حاسبة الراتب تتبع نمط **MVC** البسيط مع فصل واضح بين المنطق والعرض.

## هيكل المشروع

```
salary-calculator/
├── index.html          # الملف الرئيسي (الواجهة)
├── style.css          # أنماط التصميم
├── script.js          # المنطق والحسابات
├── ARCHITECTURE.md    # هذا الملف
├── FORMULAS.md        # الصيغ الرياضية
├── DATA-MODEL.md      # نموذج البيانات
├── DEVELOPMENT.md     # دليل التطوير
├── CONTRIBUTING.md    # دليل المساهمة
└── Changelog          # سجل التغييرات
```

## المكونات الرئيسية

### 1. **HTML (index.html)**
- الواجهة الرئيسية
- تقسيم إلى أقسام منطقية:
  - رأس الصفحة (Header)
  - بطاقات الملخص (Summary Cards)
  - المعدلات (Wage Rates)
  - الإعدادات (Settings)
  - البدلات (Allowances)
  - الإسقاطات (Projections)
  - التذييل (Footer)

### 2. **CSS (style.css)**
- نظام الألوان (CSS Variables)
- التصميم المتجاوب (Responsive Design)
- دعم المظهر الداكن (Dark Mode)
- دعم الاتجاهات (RTL/LTR)

### 3. **JavaScript (script.js)**

#### أنظمة رئيسية:
- **نظام الترجمة (i18n)**: دعم العربية والإنجليزية
- **نظام المظهر (Theme)**: Light/Dark Mode
- **نظام الحسابات (Calculations)**: حساب الراتب والبدلات
- **نظام الحفظ (Storage)**: localStorage للبيانات المحلية

#### الدوال الرئيسية:
```javascript
calc()                  // حساب الراتب والبدلات
renderAllowEditor()     // عرض محرر البدلات
updatePageLanguage()    // تحديث لغة الصفحة
toggleLanguage()        // تبديل اللغة
toggleTheme()          // تبديل المظهر
updateDynamicContent() // تحديث المحتوى الديناميكي
```

## تدفق البيانات

```
المستخدم يدخل البيانات
    ↓
oninput event → updateAllow() / calc()
    ↓
حساب القيم الجديدة
    ↓
تحديث DOM مع الترجمة
    ↓
حفظ في localStorage
    ↓
عرض النتائج على الفور
```

## نموذج البيانات

### البدلات (ALLOWANCES)
```javascript
{
  name: string,           // اسم البدل
  type: "percent" | "fixed", // نوع البدل
  value: number,         // القيمة (نسبة أو مبلغ)
  icon: emoji,           // رمز البدل
  sakan: boolean        // هل يدخل في وعاء التأمينات
}
```

### المتغيرات الأساسية
- `i_basic`: الراتب الأساسي
- `i_gosi`: نسبة التأمينات
- `i_hours`: ساعات العمل اليومية
- `i_days`: أيام العمل في الأسبوع
- `i_otMult`: معامل أجر الإضافي
- `i_eidMult`: معامل أجر العيد
- `i_raise`: قيمة العلاوة السنوية
- `i_raiseType`: نوع العلاوة (fixed/percent)
- `i_years`: مدة الإسقاط

## الحسابات الأساسية

### صافي الراتب
```
صافي الراتب = الإجمالي - التأمينات
الإجمالي = الأساسي + البدلات
التأمينات = (الأساسي + بدل السكن) × نسبة التأمينات
```

### المعدلات
```
أجر اليوم = الإجمالي ÷ 30
أجر الساعة = أجر اليوم ÷ ساعات العمل
أجر الإضافي = أجر الساعة × معامل الإضافي
```

## نظام الترجمة

- نظام `i18n` يدعم العربية والإنجليزية
- دالة `t(key)` لترجمة النصوص
- تحديث فوري عند تبديل اللغة
- حفظ اختيار اللغة في localStorage

## نظام المظهر

- متغيرات CSS للألوان
- دعم Dark Mode و Light Mode
- حفظ تفضيل المستخدم
- تغيير سلس بدون تحديث الصفحة

## الأداء والتحسينات

- استخدام localStorage للحفظ السريع
- تحديث فوري بدون تأخير
- لا توجد مكتبات خارجية (Vanilla JavaScript)
- حجم الملفات صغير وخفيف
