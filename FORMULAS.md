# الصيغ الرياضية - Formulas

## الحسابات الأساسية للراتب

### 1. حساب الراتب الإجمالي
```
الراتب الإجمالي = الراتب الأساسي + إجمالي البدلات

إجمالي البدلات = Σ البدلات
  - البدلات النسبية = الراتب الأساسي × (النسبة ÷ 100)
  - البدلات الثابتة = المبلغ المحدد
```

### 2. حساب التأمينات الاجتماعية (GOSI)
```
التأمينات = (الراتب الأساسي + بدل السكن) × (نسبة التأمينات ÷ 100)

حيث:
- نسبة التأمينات الافتراضية = 9.75%
- بدل السكن يُحدد تلقائياً (أي بدل يحتوي على "سكن" في اسمه)
```

### 3. حساب صافي الراتب
```
صافي الراتب = الراتب الإجمالي - التأمينات

صافي الراتب = (الأساسي + البدلات) - التأمينات
```

## معدلات الأجر بالساعة والیوم

### 4. أجر اليوم الواحد
```
أجر اليوم = الراتب الإجمالي ÷ 30

حيث:
- الشهر الواحد = 30 يوم (حسب الطريقة القياسية)
```

### 5. أجر الساعة الواحدة
```
أجر الساعة = أجر اليوم ÷ ساعات العمل اليومية

مثال:
إذا كان أجر اليوم = 100 ريال
وساعات العمل = 8 ساعات
إذاً أجر الساعة = 100 ÷ 8 = 12.5 ريال
```

### 6. أجر الإضافي (Overtime)
```
أجر الإضافي = أجر الساعة × معامل الإضافي

حيث:
- معامل الإضافي الافتراضي = 1.5x
- يعني: كل ساعة إضافية تُحسب بـ 1.5 من الأجر العادي
```

### 7. أجر العيد/الإجازة
```
أجر العيد = أجر الساعة × معامل العيد

حيث:
- معامل العيد الافتراضي = 2x
- يعني: ساعات العيد تُحسب بـ ضعفي الأجر العادي
```

## حسابات ساعات وأيام العمل

### 8. أيام العمل الشهرية الفعلية
```
أيام العمل الشهرية = أيام العمل بالأسبوع × 4.33 أسبوع

حيث:
- 4.33 = 52 أسبوع ÷ 12 شهر (المتوسط السنوي)

مثال:
إذا كان العمل 5 أيام بالأسبوع
أيام العمل الشهرية = 5 × 4.33 = 21.65 يوم
```

### 9. ساعات العمل الشهرية
```
ساعات العمل الشهرية = أيام العمل الشهرية × ساعات العمل اليومية

مثال:
21.65 أيام × 8 ساعات = 173.2 ساعة شهرياً
```

## الإسقاطات والتوقعات المستقبلية

### 10. الراتب بعد العلاوة السنوية (المبلغ الثابت)
```
الراتب السنة الثانية = الراتب الحالي + قيمة العلاوة

مثال:
الراتب السنة 1 = 5000 ريال
العلاوة السنوية = 300 ريال
الراتب السنة 2 = 5000 + 300 = 5300 ريال
الراتب السنة 3 = 5300 + 300 = 5600 ريال
```

### 11. الراتب بعد العلاوة السنوية (نسبة مئوية)
```
الراتب السنة الثانية = الراتب الحالي × (1 + نسبة العلاوة ÷ 100)

مثال:
الراتب السنة 1 = 5000 ريال
العلاوة السنوية = 5%
الراتب السنة 2 = 5000 × 1.05 = 5250 ريال
الراتب السنة 3 = 5250 × 1.05 = 5512.5 ريال
```

### 12. إجمالي الزيادة التراكمية
```
إجمالي الزيادة = Σ (صافي الراتب في كل سنة - صافي الراتب الأساسي) × 12

شرح:
- نحسب الفرق في الراتب الشهري
- نضربه × 12 (عدد الأشهر)
- نجمع النتائج لجميع السنوات
```

## نسبة البدلات

### 13. نسبة إجمالي البدلات من الراتب الأساسي
```
نسبة البدلات % = (إجمالي البدلات ÷ الراتب الأساسي) × 100

مثال:
الراتب الأساسي = 5000 ريال
البدلات = 1500 ريال
نسبة البدلات = (1500 ÷ 5000) × 100 = 30%
```

## أمثلة عملية

### مثال شامل:
```
البيانات المدخلة:
- الراتب الأساسي: 5000 ريال
- بدل سكن: 25% = 1250 ريال
- بدل مواصلات: 10% = 500 ريال
- نسبة التأمينات: 9.75%
- ساعات العمل: 8 ساعات
- أيام العمل: 5 أيام

الحسابات:
1. إجمالي البدلات = 1250 + 500 = 1750 ريال
2. الراتب الإجمالي = 5000 + 1750 = 6750 ريال
3. وعاء التأمينات = 5000 + 1250 = 6250 ريال
4. التأمينات = 6250 × 0.0975 = 609.375 ريال
5. صافي الراتب = 6750 - 609.375 = 6140.625 ريال

المعدلات:
6. أجر اليوم = 6750 ÷ 30 = 225 ريال
7. أجر الساعة = 225 ÷ 8 = 28.125 ريال
8. أجر الإضافي = 28.125 × 1.5 = 42.1875 ريال
9. أجر العيد = 28.125 × 2 = 56.25 ريال

أيام وساعات العمل:
10. أيام العمل الشهرية = 5 × 4.33 = 21.65 أيام
11. ساعات العمل الشهرية = 21.65 × 8 = 173.2 ساعة
```

## ملاحظات مهمة

1. **دقة الكسور العشرية**: يتم الاحتفاظ بـ 4 كسور عشرية عند الحسابات
2. **التقريب في العرض**: تُعرض النتائج برقمين عشريين للوضوح
3. **شهر = 30 يوم**: معيار دولي موحد للحسابات المحاسبية
4. **التأمينات على وعاء محدد**: تُحسب فقط على (الأساسي + بدل السكن)
5. **البدلات النسبية**: تُحسب دائماً من الراتب الأساسي فقط
