# 🧮 المعادلات الحسابية — Formulas

## المعادلة الأساسية للراتب

```
إجمالي البدلات = Σ (لكل بدل)
    البدل النسبي  = الأساسي × (النسبة ÷ 100)
    البدل الثابت  = القيمة كما هي

الراتب الإجمالي = الأساسي + إجمالي البدلات

التأمينات (GOSI) = (الأساسي + بدل السكن) × نسبة التأمينات

الراتب الصافي = الراتب الإجمالي − التأمينات
```

> **ملاحظة مهمة:** التأمينات تُحسب على **الأساسي + بدل السكن فقط** (وعاء الاشتراك في GOSI)، وليس على كامل الراتب. النظام يتعرّف على بدل السكن تلقائياً من اسمه.

### مثال

| البند | القيمة |
|------|--------|
| الأساسي | 10,000 |
| بدل سكن 25% | 2,500 |
| بدل مواصلات 10% | 1,000 |
| بدل طبيعة عمل 15% | 1,500 |
| بدل تأهيل 5% | 500 |
| **إجمالي البدلات** | **5,500** |
| **الإجمالي** | **15,500** |
| التأمينات = (10,000 + 2,500) × 9.75% | 1,218.75 |
| **الصافي** | **14,281.25** |

## معدّلات الأجر

```
أجر اليوم   = الراتب الإجمالي ÷ 30
أجر الساعة  = أجر اليوم ÷ ساعات العمل اليومية
أجر الإضافي = أجر الساعة × معامل الإضافي (افتراضي 1.5)
أجر العيد   = أجر الساعة × معامل العيد (افتراضي 2)

أيام العمل الشهرية  = أيام العمل الأسبوعية × 4.33
ساعات العمل الشهرية = أيام العمل الشهرية × ساعات العمل اليومية
```

> 4.33 = متوسط عدد الأسابيع في الشهر (52 أسبوع ÷ 12 شهر).

### مثال (إجمالي 15,500، 8 ساعات، 5 أيام)

| المعدّل | القيمة |
|--------|--------|
| أجر اليوم | 516.67 |
| أجر الساعة | 64.58 |
| أجر الإضافي/ساعة | 96.88 |
| أجر العيد/ساعة | 129.17 |
| أيام العمل/شهر | 21.65 |
| ساعات العمل/شهر | 173 |

## العلاوة السنوية والتوقّع

كل سنة، تُضاف العلاوة للأساسي، ثم **يُعاد حساب كل شيء** (لأن البدلات النسبية والتأمينات ترتفع مع الأساسي):

```
العلاوة الثابتة:  الأساسي_السنة+1 = الأساسي_السنة + قيمة العلاوة
العلاوة النسبية:  الأساسي_السنة+1 = الأساسي_السنة × (1 + النسبة ÷ 100)
```

لكل سنة:
```
صافي_السنة = computeFor(الأساسي_السنة, نسبة التأمينات).net
الزيادة التراكمية = صافي_السنة − الصافي_الحالي
```

### مثال (أساسي 10,000، علاوة 500 ثابتة/سنة)

| السنة | الأساسي | الصافي |
|------|---------|--------|
| الآن | 10,000 | 14,281.25 |
| سنة 1 | 10,500 | 14,995.31 |
| سنة 5 | 12,500 | 17,851.56 |
| سنة 10 | 15,000 | 21,421.88 |

العلاوة الثابتة 500 على الأساسي تُنتج زيادة صافية ~714/سنة، لأنها ترفع البدلات النسبية والوعاء أيضاً.

## ملاحظات الدقة

- كل الحسابات بأرقام عشرية كاملة، والعرض يُقرّب لخانتين.
- العملة والأرقام تُعرض بالنظام اللاتيني (123) عبر `toLocaleString("en-US")`.
- النسب تُدخل كأعداد مئوية (25 تعني 25%) وتُقسم على 100 داخلياً.
