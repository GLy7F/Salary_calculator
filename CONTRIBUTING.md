# دليل المساهمة - Contributing Guide

شكراً لاهتمامك بالمساهمة في حاسبة الراتب! نرحب بجميع أشكال المساهمة.

## أنواع المساهمات المقبولة

✅ إصلاح الأخطاء  
✅ إضافة ميزات جديدة  
✅ تحسين الأداء  
✅ تحسين التوثيق  
✅ ترجمات جديدة  
✅ تحسين واجهة المستخدم  
✅ تقارير الأخطاء  

## معايير جودة الكود

### التسمية
- استخدم camelCase للمتغيرات والدوال
- أسماء واضحة ووصفية

### التنسيق
- استخدم مسافات ومحاذاة صحيحة
- تعليقات مفيدة عند الحاجة

## معايير الترجمة

عند إضافة نصوص جديدة أضفها إلى i18n:
- العربية والإنجليزية معاً
- نصوص واضحة وموجزة

## تحديث التوثيق

عند إضافة ميزة جديدة، حدث:
1. ARCHITECTURE.md
2. DATA-MODEL.md  
3. FORMULAS.md
4. Changelog
